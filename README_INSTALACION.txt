SISTEMA DE SORTEO - V2 PREMIUM

Incluye:
- Landing con números del 00 al 99.
- Selección múltiple de números.
- Envío a WhatsApp.
- Dashboard privado.
- Registro manual de participantes.
- Cambio de estado: disponible, reservado, vendido y bloqueado.
- Botón Eliminar alineado junto a Cambiar en escritorio.
- Paleta premium negro + dorado.
- Base de datos MySQL.

INSTALACIÓN EN XAMPP
1. Copia la carpeta sorteo_landing_dashboard_v2 dentro de:
   C:\xampp\htdocs\

2. Abre XAMPP y activa Apache y MySQL.

3. Entra a phpMyAdmin:
   http://localhost/phpmyadmin

4. Crea una base de datos llamada:
   sorteo_db

5. Importa el archivo:
   database.sql

6. Abre el proyecto:
   http://localhost/sorteo_landing_dashboard_v2/

DASHBOARD

CONFIGURACIÓN
Edita config.php para cambiar:
- Nombre del sorteo.
- Número de WhatsApp.
- Datos de conexión MySQL.

IMPORTANTE
En hosting/cPanel, cambia DB_USER, DB_PASS y DB_NAME según los datos reales de tu hosting.
